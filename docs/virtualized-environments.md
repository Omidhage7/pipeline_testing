# Virtualized Environment Standard

Application workloads are required to run in isolated virtualized environments.

The infrastructure standard includes:

- Dedicated virtual machines or containerized environments
- Isolation between application workloads
- Separate development, testing, and production environments
- Controlled allocation of compute resources
- Environment-specific configuration

Production workloads must not run directly on unmanaged physical infrastructure.

Virtualized or containerized environments must be used to provide workload isolation and controlled infrastructure management.
